HOW TO DOWNLOAD:
---------------
1. Visit one of the links above
2. Click "Free Download" button
3. Select resolution: 1920x1080 (Full HD) recommended
4. Download the MP4 file
5. Rename it to: hero-background.mp4
6. Place it in this folder (public/videos/)

TIPS FOR BEST RESULTS:
---------------------
- Choose a video with slow, ambient movement
- Avoid videos with too much action or bright flashes
- Darker videos work better (the overlay will add contrast)
- Videos 10-30 seconds long loop nicely
- 1920x1080 resolution is ideal for web

The website will show the fallback image (hero-lighting.png) until the video loads.
